# Controls

- WASD: Movement
- Left click: select
- Movement + SHIFT: speed x2.
- F: focus the camera around the selected object.
- Mouse wheel + Alt: Zoom

- Esc: Close the engine

ATENTION! Drag and drop for textures work ONLY before creating a geometry.
